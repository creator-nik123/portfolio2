Game of War

A "Game of War" card game built with HTML, CSS, and JavaScript uses an external API to shuffle and draw cards. Players click a button to draw a card, and each round’s winner is determined by comparing card values, with scores tracked and displayed in real time until all cards are used or a set score is reached.

Notes App

A notes app built with React, JavaScript, and Firebase allows users to create, edit, and delete notes. Firebase is used for authentication and as a real-time database to store and sync notes across devices. Users can organize their notes in categories, with updates reflected immediately due to Firebase's real-time capabilities.
